Async Chat Widget Loading

Testing asynchronous loading of the chat widget

Loading Methods:

1. Async Script Loading

<script> // Load the widget asynchronously (function() { const script = document.createElement('script'); script.src = './dist/chat-widget.js'; script.type = 'module'; script.async = true; script.onload = function() { // Initialize after loading window.ChatWidget.init({ primaryColor: '#28a745', title: 'Async Chat', position: 'bottom-left' }); }; document.head.appendChild(script); })(); </script>

2. Dynamic Import (Modern)

<script type="module"> async function loadChatWidget() { try { const { ChatWidget } = await import('./dist/chat-widget.js'); ChatWidget.init({ primaryColor: '#dc3545', title: 'Modern Chat' }); } catch (error) { console.error('Failed to load chat widget:', error); } } loadChatWidget(); </script>

3. Data Attribute Auto-initialization

<script type="module" src="./dist/chat-widget.js" data-auto-load="true" data-primary-color="#6f42c1" data-title="Auto Chat" data-position="bottom-right"> </script>

üöÄ How to Deploy to Cloudflare Pages

Option 1: Direct Upload

  1. Make sure you've run npm run build (‚úÖ Done - file exists)
  2. Go to Cloudflare Dashboard
  3. Navigate to Pages ‚Üí Your project (cw-vanilla)
  4. Upload or drag & drop the dist/chat-widget.umd.cjs file
  5. Make sure it's uploaded to the root directory (not in a subfolder)

Option 2: Git Integration

  1. Commit your changes: git add . && git commit -m "Update widget build"
  2. Push to your repository: git push origin main
  3. Cloudflare Pages will automatically deploy from your git repository
  4. Set build command to: npm run build
  5. Set output directory to: dist

Verify Deployment

Once deployed, test these URLs:

  • Main site: https://cw-vanilla.pages.dev/
  • Widget file: https://cw-vanilla.pages.dev/chat-widget.umd.cjs

Embedding Code (After Deployment)

Once your widget is deployed to Cloudflare Pages, use this code:

Custom Configuration

  src="https://cw-vanilla.pages.dev/chat-widget.umd.cjs"
  data-auto-load="true"
